# 💍 Narasimha & Reesheka — Wedding Invitation

A traditional Telugu wedding invitation web page with beautiful Ghibli-style illustrations.

---

## 📋 What You Need to Upload to GitHub

Four files go in the repository (all in the same root folder):

1. **`index.html`** — the webpage
2. **`mandapam.png`** — couple at the mandapam
3. **`pradakshina.png`** — couple walking around the homa gundam
4. **`arundhati.png`** — couple pointing at Arundhati star

---

## 🚀 How to Host on GitHub Pages (Step-by-Step)

### Step 1: Create a GitHub Account
Sign up at [github.com](https://github.com) if you don't have an account.

### Step 2: Create a New Repository
1. Click the **+** icon (top right) → **New repository**
2. Repository name: `wedding-invite` (or any name)
3. Set to **Public** (required for free GitHub Pages)
4. Check **Add a README file**
5. Click **Create repository**

### Step 3: Upload ALL Four Files Together
1. Click **Add file** → **Upload files**
2. Drag all four files into the upload area:
   - `index.html`
   - `mandapam.png`
   - `pradakshina.png`
   - `arundhati.png`
3. Scroll down and click **Commit changes**

> ⚠️ **CRITICAL:** All four files must be in the SAME folder (the root of the repo). The HTML expects the PNG images to be right next to it.

### Step 4: Enable GitHub Pages
1. Go to repo **Settings** (top right tab)
2. In the left sidebar, click **Pages**
3. Under "Source", select **Deploy from a branch**
4. Branch: **main**, Folder: **/ (root)**
5. Click **Save**

### Step 5: Get Your URL
After 1–2 minutes, your invitation will be live at:

```
https://YOUR-USERNAME.github.io/wedding-invite/
```

Share via WhatsApp, email, social media! 🎉

## ✏️ How to Edit Details Later

Open `index.html` in any text editor and search for:

| What to Change | Find This in the File |
|----------------|------------------------|
| Couple's names | `Narasimha` and `Reesheka` |
| Telugu names | `నరసింహ & రీషికా` |
| Wedding date | `2026-06-28T02:04:00+05:30` (countdown timer) |
| Venue addresses | Inside the `events-section` |
| Google Maps links | Search for `maps.app.goo.gl` and replace |
| Parents' names | Inside the `families-section` |
| Contact numbers | `+91 98499 33495` at the bottom |
| Blessing text | `శుభమస్తు • శ్రీరస్తు • అవిఘ్నమస్తు` |

After editing, just re-upload `index.html` to GitHub — the live site updates automatically within a minute.

### To swap any image later:
Just upload a new PNG with the same exact filename (`mandapam.png`, `pradakshina.png`, or `arundhati.png`) — it'll overwrite the old one and the website updates immediately.
